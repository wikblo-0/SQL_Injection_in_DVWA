This repository stores the payloads used in the penetration testing of the SQL Injection vulnerability in Damn Vulnerable Web Application (DVWA).

- payloads_for_low_and_high_security_levels.sql stores payloads used on the low and high security levels.

- payloads_for_medium_security_level.sql stores payloads used on the medium security level.

- shell_script_for_low_security_level.sh contains an executable shell script used for automating attacks on the low security level.

- shell_script_for_medium_security_level.sh contains an executable shell script used for automating attacks on the medium security level.

- shell_script_for_high_security_level.sh contains an executable shell script used for automating attacks on the high security level.

You can read more about the attack in the blog linked below, and there you will also find YouTube videos showing a screencast of each attack.
